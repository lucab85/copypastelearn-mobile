## ADDED Requirements

### Requirement: All touch targets SHALL be at least 44px
- **WHEN** any touchable element is rendered
- **THEN** its hit area SHALL be at least 44x44 points

### Requirement: All interactive elements SHALL have accessibility labels
- **WHEN** a screen reader focuses on an interactive element
- **THEN** it SHALL announce a meaningful label describing the action

### Requirement: Text SHALL support dynamic type
- **WHEN** user changes system font size
- **THEN** app text SHALL scale proportionally
